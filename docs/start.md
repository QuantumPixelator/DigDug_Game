# How to Run the Game

This game uses ES6 modules, and requires it to be served from a web server due to browser security restrictions.

## Instructions

1.  **Install Node.js:** If you don't have Node.js installed, download and install it from [https://nodejs.org/](https://nodejs.org/).

2.  **Start the server:** Open a terminal or command prompt in the game's root directory and run the following command:
    ```
    node server.js
    ```

3.  **Open the game:** Open your web browser and navigate to the following address:
    ```
    http://server-ip-address:8000
    ```

That's it! The game should start right away.
